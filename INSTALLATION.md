# Altera APAC - Term License Revenue Calculator
## Installation Guide

### Quick Start

This is a standalone web application that requires **no installation**. Simply open the HTML file in your web browser.

---

## How to Use

### Option 1: Direct Open (Recommended)
1. Extract all files to a folder on your computer
2. Double-click `index.html`
3. The calculator will open in your default web browser
4. Start entering contract information

### Option 2: Run from Any Location
1. Extract the files anywhere on your computer
2. Right-click `index.html`
3. Select "Open with" and choose your preferred browser:
   - Google Chrome
   - Microsoft Edge
   - Firefox
   - Safari

---

## Troubleshooting

**Issue**: Calculator doesn't open
- **Solution**: Make sure all files are in the same folder
- **Solution**: Try right-clicking and selecting "Open with" → Browser

**Issue**: Logo doesn't appear
- **Solution**: Ensure `logo.png` is in the same folder as `index.html`

**Issue**: Styling looks wrong
- **Solution**: Ensure `styles.css` is in the same folder as `index.html`

**Issue**: Calculator doesn't work
- **Solution**: Ensure `calculator.js` is in the same folder as `index.html`
- **Solution**: Check if JavaScript is enabled in your browser

**Issue**: Can't export to CSV
- **Solution**: Check your browser's download settings
- **Solution**: Allow downloads from local files in your browser
